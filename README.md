React.js x Node.js
========

Command
---

~~~
npm run clean
~~~
Remove : build/ public/bundle.js

~~~
npm run build
~~~
Create : build/ public/bundle.js

~~~
npm run start
~~~
Execute : NODE_ENV=production

~~~
npm run development
~~~
Execute : NODE_ENV=development with 'react-hot-loader' (in fix)
